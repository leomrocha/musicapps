(1) Introduction

  MIDITrail is a MIDI player which provides 3D visualization of
  MIDI datasets. MIDITrail supports SMF format 0/1, and multiple
  MIDI ports.

(2) System requirement

  OS: Windows 2000 / XP / Vista / 7 / 8
  VIDEO: Support for DirectX 9.0c graphics

(3) Quick Start

  1. Drop Standard MIDI file(*.mid) to the window.
  2. Push space key to play/pause, and push ESC key to stop.

(4) User manual

  Please check "Help" -> "Manual...".

(5) About DirectX End-User Runtime

  If you get "d3dx9_**.dll was not found" error, please install the 
  latest version of "DirectX End-User Runtime".

  Microsoft Download Center
  http://www.microsoft.com/download/en/default.aspx

(6) NOTICE for Windows 2000 users

  If you are using Windows 2000, you must move "msvc*.dll" files to 
  the folder "MIDITrail.exe" exists from "Microsoft.VC90.CRT" folder,
  before executing "MIDITrail.exe".
  If you are using Windows XP / Vista / 7, do not move DLL files.
