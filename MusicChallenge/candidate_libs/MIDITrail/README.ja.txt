(1) はじめに

  MIDITrailはMIDIデータを三次元可視化するMIDIプレーヤーです。
  標準MIDIファイルFormat0/1、および複数ポート出力に対応しています。

(2) 動作環境

  OS： Windows 2000 / XP / Vista / 7 / 8
  VIDEO: DirectX 9.0c に対応したグラフィックチップが必要です。

(3) クイックスタート

  1. 標準MIDIファイル(*.mid)をウィンドウにドロップしてください。
  2. スペースキーで演奏開始／一時停止、ESCキーで演奏停止します。

(4) ユーザーマニュアル

  "Help" -> "Manual..."を参照してください。

(5) DirectX エンドユーザランタイムについて

  MIDITrail実行時に「d3dx9_**.dllが見つからない」というエラーが発生
  した場合は、マイクロソフト社のWebページにアクセスして、最新の
  「DirectX エンドユーザーランタイム(DirectX End-User Runtime)」を
  インストールしてください。

  Microsoft: DirectX (ダイレクトエックス) 最新ダウンロード/統合情報
  http://www.microsoft.com/japan/directx/default.mspx

(6) Windows 2000をお使いの方へ

  Windows 2000で使用する場合は、MIDITrail実行前に、Microsoft.VC90.CRT
  フォルダにある msvc*.dll ファイルを MIDITrail.exe と同じフォルダに
  移動してください。
  Windows XP / Vista / 7 の場合はDLLファイルを移動しないでください。
